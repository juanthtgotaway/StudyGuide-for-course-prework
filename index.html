Prework Study Guide

HTMl

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The h1 element is for headings
  • The h2 element is used for headings but smaller subjects
  • The li element is used for lists

CSS

  • A Margin indicates how much space we want around the outisde of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS adds the physical colors, spacing and margins for the entire website.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git pull origin main: pulls original document/ or most recent
  • git add --all: allows all files to be added to the terminal
